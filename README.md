# python_30_day_Challenge

Solutions and practice exercises for the 30-day Python challenge.

## Usage
Run an example file with Python:

```bash
python day1.py
```

## Notes
- Each day is a separate script. Feel free to convert to modules as needed.
- Add tests under a `tests/` directory if desired.

## Tests

Run tests with pytest (install dev requirements first):

```bash
pip install -r requirements-dev.txt
pytest
```
